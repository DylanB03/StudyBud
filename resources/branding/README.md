# Branding assets

`forge.config.ts` references `resources/branding/icon` without an extension, so
Electron Packager resolves the correct format per target:

- `icon.icns` — macOS `.app` bundle icon.
- `icon.ico` — Windows installer / `.exe` icon.
- `icon.png` — Linux builds (and fallback).

The committed files are **placeholders** generated by
`scripts/generate-placeholder-icon.mjs`. Replace them with real artwork by
running that script with a different design, or by dropping hand-authored
assets with the same names.

If you have a single source PNG (1024x1024, RGBA) you want to convert:

1. Replace the `makePixels()` body in `scripts/generate-placeholder-icon.mjs`
   with code that loads your PNG into a `Buffer` of RGBA bytes.
2. Run `node scripts/generate-placeholder-icon.mjs` to regenerate all three
   files in the required formats/sizes.
